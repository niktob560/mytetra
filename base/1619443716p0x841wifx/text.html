АЦП последовательного типа

В таких АЦП обычно используется только один компаратор, который сравнивает входной сигнал с эталонным напряжением. Соответствующие уровни эталонного напряжения формируются с помощью ЦАП. В состав таких АЦП входит и ЦАП. Входные коды для ЦАП в этом случае генерирует либо двоичный счётчик, либо регистр последовательного приближения.

АЦП последовательного счёта


В состав такого АЦП будет входить генератор тактовых импульсов (ГТИ), который определяет частоту импульсов, поступающих на вход счётчика, 3И-НЕ, который формирует лог. 0 только когда на всех входах лог. 1, двоичный счётчик, ЦАП, ИОН и регистр (RG), компаратор


По сигналу "запуск" схема управления формирует сигнал сброса для счетчика, на выходе счетчике выставляются нули, следовательно на выходе ЦАП устанавливается 0В, которое поступает на инвертирующий вход компаратора. Компаратор и схема управления устанавливают логическую единицу на входах элемента 3И-НЕ, следовательно на выходе элемента будут появляться тактовые импульсы с выхода ГТИ (но в противофазе). Тактовый импульс поступает на вход C(CLK) счётчика, следовательно на выходе устанавливается двоичный код на единицу больше предыдущего. Преобразование заканчивается когда напряжение на выходе ЦАП устанавливается больше чем входное.
