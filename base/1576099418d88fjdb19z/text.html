Интерфейсы МЭМС-датчиков и цифровых микросхем


Большинство МЭМС-датчиков ориентировано на подключение к микроконтроллеру посредством последовательного цифрового интерфейса


В отличие от параллельного интерфейса, когда обмен данными с МК происходит одновременно по нескольким линиям связи, последовательный интерфейс подразумевает использование минимального кол-ва линий связи с МК

Parralel

+======+ +=======+

| Датчик |------------| MCU |

| |------------| |

| |------------| |

| |------------| |

| |------------| |

| |------------| |

+======+ +=======+


Serial

+======+ +=======+

| Датчик |-----------| MCU |

| | | |

| | | |

| | | |

+======+ +=======+



Такой способ передачи позволяет уменьшить кол-во выводов датчика и общие габаритные размеры корпуса микросхемы, в которую он был помещён


Одни из наиболее популярных протоколов обмена последовательного интерфейса:

->1-Wire

->SPI

->I2C
