Плюсы и минусы -АЦП


Плюсы:

  1. Простота схемы
  2. Высокая точность преобразования
  3. Усредение выходного сигнала
  4. Изменяемая разрядность


Минусы:34

  1. Высокая точность достигается только при медленном изменении входного сигнала
  2. Предпоследний по скорости преобразования (быстрее чем интегрирующий)
  3. Необходимость подключения высокочастотного ГТИ
  4. Необходимость изолировать кварцевый резонатор для исключения влияния помех


Аналоговый мультиплексор

Используется для коммутации аналоговых сигналов

Подразумевает наличие нескольких входных сигналов



Аналоговые мультиплексоры используют реле, оптроын, герконы, но чаще всего - МОП-ключи.


Основные параметры МОП-ключа:

  1. Сопротивление канала в открытом и закрытом состоянии (обычно и )
  2. Разность сопротивлений каналов
  3. Ток утечки по входу и выходу
  4. Время переключения каналов (обычно и )
  5. Температурный дрейф параметров
