Параметризация геометрических моделей


Геометрия может быть 2-х типов

  • Конструктивная твердая модель (CSG)
  • Граничное представление (BR)



Общее представление построения объекта:

  1. Базовый объект в пространстве
  2. Создание эскиза
  3. Нанесение размеров и зависимостей
  4. Создание 3D объекта


Основные вехи в создании математических основ современных геометрических моделей

  • Станки с ЧПУ в MIT в 50-х годах
  • Создание скульптурных поверхностей для Citroen
  • ...


Параметрическая геометрическая модель[ПГМ] (научное определение) - представление множества однотипных геометрических моделей[ГМ] в виде функции от независимых величин - параметров


Параметрическая геометрическая модель[ПГМ] (обычное определение) - геометрическая модель[ГМ], управляемая параметрами с сохранением заданных отношений между элементами ГМ


/*болт с размерами жпг*/


Достоинства ПГМ:

  • Очень быстрая параметрическая модификация готовой модели
  • Повторное использование разработанных моделей, возможность организации библиотек типовых параметрических элементов
  • Упрощение подготовки чертежей типовых изделий
  • Всокая точность получаемых ГМ
  • Автоматический контроль и учет заданных отношений


Недостатки ПГМ:

  • Требует дополнительных действий от пользователя для построения параметрической модели
  • Требует дополнительного механизма автоматического построения экземпляра модели (решателя)


Основные области применения ПГМ:

  • Типовое конструирование (новая конструкция - старая с новыми параметрами)
  • Вариантное конструирования (поиск оптимальных параметров конструкции)
  • Моделирование кинематики механизмов (параметризуется движение деталей механизма)
  • Параллельный инжиниринг (ещё неизвестные параметры взаимосвязанных деталей механизма заменяются переменными)


Параметризация:

  • Процесс создания ПГМ
  • Результат создания ПГМ
  • Процесс расчета ПГМ
  • Технология ПГМ

Технологии параметризации:

  • Процедурная параметризация (пишем код) [Autocad, OpenSCAD]
  • Параметризация по "истории" построения (иерархическая) [Inventor, Pro/E]
  • Процедурно-иерархическая параметризация [FreeCAD, SprutCAD]
  • Объектно-ориентированная параметризация [FreeCAD, Inventor]
  • Вариационная жесткая параметризация
  • Вариационная мягкая параметризация


Процедурная параметризация

Достоинства:

  • Высокая скорость расчета процедурных ПГМ
  • Возможность реализации инженерных расчетов в процессе синтеза
  • Возможность реализации структурного синтеза

Недостатки:

  • Необходимо программирование
  • Высокая трудоёмкость создания и редактирования ПГМ
  • Значительное редактирование модели требует полного переписывания программы
  • Необходима квалификация программиста
  • Программирование - неестественно для конструктора

Применение:

  • Разработка библиотек ПГМ простых типовых ГМ
  • Разработка специализированных САПР на основе универсальных САПР


Иерархическая параметризация

Есть история(дерево) построений

Достоинства:

  • Не требуется квалификация программиста
  • Высокая скорость расчета ПГМ
  • Простая организация структурного синтеза

Недостатки:

  • Пользователь должен планировать порядок построения
  • Высокая трудоемкость редактирования модели

Применение:

  • Твердотельное 3D геометрическое моделирование


Объектно-ориентированная параметризация

Модель собирается из активных интеллектуальных объектов - конструктивных элементов (фичерсов) таких как отверстие, фаска, скругление, ребро жесткости и т.п.


Достоинства:

  • Высокая скорость расчета
  • Интеллектуальная адаптивность фичерсов к изменениям
  • Фичермы понятийно близки конструктору и технологу

Недостатки:

  • Пользователь должен сам планировать порядок построения фичерсов
  • Ограниченность набора фичерсов

    • Применение:

  • Твердотельное 3D геометрическое моделирование совместно с иерархической параметризацией
