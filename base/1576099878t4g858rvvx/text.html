1-Wire

Изначально 1-Wire был разработан DallasSemiconductor и в основонм использовался в микросхемах, выпускаемых ими


1-Wire используют для двустороннего обмена данными всего 1 линию и общую GND


DQ должна быть подключена к линии питания через резистор 4.7КОм


Ucc

Master /\ Slave

======+ || +======+

|MCU DQ |<----++---->| DQ MCU|

| GND |--| |--| GND |

| | | | | |

======+ | | +======+

-+- -+-


Пример схемы подключения по 1-Wire

Ucc

Master /\ Slave

+======+ || +======+

|MCU DQ |<-----++----->| DQ MCU|

| GND |--| | |--| GND |

| | | | | | |

+======+ --- | --- +======+

| Slave

| +=======+

-----------| DQ MCU|

|--| GND |

| | |

--- +=======+


Такую схему включения называют схемой с "паразитным" питанием

Скорость обмена может достигать до 100КБит/с

Используя этот интерфейс можно строить сеть из большого кол-ва устройств (до 250). Такая сеть называется MicroLAN

При 3-х проводном соединении (добавляем VCC) можно осуществлять обмен данными на расстоянии до 300 метров


Каждое ведомое ус-во имеет свой собственный уникальный 64-х разрядный(8 байт) адрес, который содержит 8 бит(1 слово) идентификатора семейства, 48 бит(6 слов) - сам адрес и ещё 8 бит(1 слово) - контрольная сумма (CRC) всех предыдущих бит

[crc][-][a][d][d][r][-][id]


код идентификатора семейства(id) указывает на тип устройства, например:


0x04(0b00000100) - энергонезависимая ОЗУ с часами, таймером и будильником DS2404

0x06(0b00000110) - ОЗУ DS1993

0x10(0b00010000) - датчик температуры DS1820, DS18B20

0x12(0b00010010) - EEPROM ПЗУ на 1Кб EEPROM и 2-хканально адресуемый ключ DS2406

0x24 - DS2415

etc


Типы памяти:

ПЗУ

ОЗУ

FLASH

EEPROM (больше циклов r/w чем у FLASH)


помимо цифровых датчиков и микросхем 1-Wire используют электронные ключи iButton

Обмен по этому интерфейсу отличается от SPI

1-Wire асинхронный, полудуплексный

Протокол последовательной передачи данных также называется 1-Wire, в котором используются специальные команды, которые содержат код команды, адрес ус-ва и данные

Двоичный код передаётся побайтно начиная с младшего разряда

для передачи бита используются временные интервалы, тайм-слоты

Тайм-слоты - определённая ограниченная по времени последовательность смены уровней сигнала на DQ

(master жмёт линию в 0, ждёт 15мкс и говорит значение)
