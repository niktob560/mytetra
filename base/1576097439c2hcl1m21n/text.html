Полнофакторный эксперимент


Иногда для сложных тех.систем неизвестны мат.модели. В таких случаях каким-либо образом пытаются найти


Модель "Чёрный ящик":

Обычно вводят:

->Ф-цию отклика y

->Входные параметры X1,X2,Xn

->Шумы W1,W2,Wn


Ф-ция отклика:

y(x1,x2,xn) = B0 + Bi * sum i 1..k (Xi) + Bij * sum i,j 1..k i!=j (Xi*Xj) + Biju * sum i,j,u 1..k i!=j!=u (Xi * Xj * Xu) + ...



Планирование эксперимента - процедура выбора числа и условия проведения опытов, необходимых и достаточных для получения мат.модели процесса


Bi = (1 / N) * sum i 1..N (Xi * Yi)

Где N - кол-во измерений

Bij = (1/N) * sum i,j 1..N i!=j (XiXj * YiYj)

Biju = (1/N) * sum i,j,u 1..N i!=j!=u (XiXjXu * YiYjYu)
